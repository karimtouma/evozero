Why it’s fast

the core

Genetic programming spends its time evaluating candidate expressions. evozero stores each as a padded postfix program and evaluates the whole population against the whole dataset as one batched tensor operation on the GPU.

The evolutionary bookkeeping stays on the CPU; only the expensive fitness pass goes to the GPU. Linear scaling, island migration, stagnation restarts and constant optimization keep the search from stalling. And import evozero never imports torch — export to LaTeX and NumPy works with no GPU at all.

Five lines to a formula

quickstart
import numpy as np
from evozero import SymbolicRegressor

X = np.random.randn(500, 2)
y = X[:, 0] ** 2 + X[:, 0] * X[:, 1] + np.sin(X[:, 1])

model = SymbolicRegressor(device="auto", random_state=0).fit(X, y)
print(model.best_equation_)   # x0*(x0 + x1) + sin(x1)
print(model.to_latex())        # -> LaTeX for your paper
